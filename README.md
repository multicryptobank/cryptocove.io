# Crypto Cove Project

Set of simple tools to streamline all your crypto operations.

## Intro

This project aims at providing a bunch of crypto utilities with the following characteristics:


- Minimalistic
- Simple
- No KYC
- Offline (in most cases)
- Multiple networks
- Integration with several wallets (e.g. Electrum, Sparrow)
- Curated list of Smart Contracts
- Zero software and/or hardware dependencies
- Flexibility / Customisation


## Getting Started
  Download the latest release and unzip it to any folder of your preference.
  
Read the README file under the 'doc' folder.
  
Enjoy.
